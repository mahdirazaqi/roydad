# Roydad

[Roydad](https://github.com/mahdirazaqi/roydad) headless cms for sell event tickets.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
